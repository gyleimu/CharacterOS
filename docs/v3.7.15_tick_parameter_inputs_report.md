# CharacterOS V3.7.15 Tick Parameter Inputs Report

## Summary

This update adds temporary manual inputs for Continuous Tick parameter-network observation.

The new inputs are:

```text
fatigue
sleepDebt
```

They affect only the `parameterNetwork` trace.

They do not persist to character state.
