## 1. 新增节点类型

| Node type | Source | Builder function | Count |
|-----------|--------|-----------------|-------|
| `desire` | `deriveDesires(needs)` — derived | `buildDesireNode` | needs.length (filtered ≥0.15) |
| `behavior_bias` | `deriveBehaviorBiases(coordinate, desires)` — derived | `buildBehaviorBiasNode` | 4 biases (filtered ≥0.05) |
| `temporal_process` | `V3_TICK_PHASES` registry metadata — derived | `buildTemporalProcessNode` | 17 (all phases) |
| `internal_state_variable` | `InternalStateFieldSnapshot` (optional) — derived | `buildInternalStateVariableNode` | 46 (if ISF provided) |

### 新增边类型

| Edge type | From | To | Weight |
|-----------|------|-----|--------|
| `drives_desire` | need → desire | desire | desire.intensity |
| `biases_behavior` | top desire → behavior_bias | behavior_bias | bias.likelihood |
| `temporal_transition` | temporal_process N → N+1 | temporal_process | 1.0 (structural) |

### Desire nodes

```
need → drives_desire → desire
```

- 每个 need 映射到一个 desire（通过 deriveDesires）
- weight = desire.intensity, clamped [0, 1]
- metadata: intensity, sourceNeedId

### Behavior bias nodes

```
desire → biases_behavior → behavior_bias
```

- 从 state.coordinate + desires 派生 4 个 biases
- top desire → each bias 有一条边
- weight = bias.likelihood, clamped [0, 1]
- metadata: likelihood, rationale, tendency

### Temporal process nodes

```
temporal_process → temporal_transition → temporal_process
```

- 来自 V3_TICK_PHASES registry metadata（17 phases）
- weight = writes.length / 10, clamped [0, 1]
- 16 temporal_transition edges（phase 1→2, 2→3, ...）
- metadata: processId, phaseNumber, reads[], writes[]

### Internal state variable nodes

```
ISV → derived_from → personality_core
```

- 可选：通过 `options.internalStateField` 提供
- weight = variable.currentValue
- mass = variable.homeostaticPressure
- metadata: domain, currentValue, homeostaticPressure, deviationFromBaseline
