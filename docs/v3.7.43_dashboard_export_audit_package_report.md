# CharacterOS V3.7.43 Dashboard Export Audit Package Report

## Summary

This update connects the character audit package export to the Dashboard.

V3.7.42 upgraded the export API so a character export includes:

```text
state
adjustment history
history summary
governance trace
```

V3.7.43 makes that export reachable from the operator interface.

## Updated Dashboard

```text
src/components/character/PhysicsDashboard.tsx
```

New control:

```text
导出审计包
```

The button:

```text
calls /api/characters/[characterId]/export
downloads a formatted JSON file
uses character id and export timestamp in the filename
updates Dashboard status on success or failure
```

## Export Filename

Generated filename pattern:

```text
<characterId>_characteros_export_<timestamp>.json
```

The character id is sanitized before being used in the file name.

## Architecture Meaning

CharacterOS now has a basic operator-grade export path:

```text
Dashboard
↓
export API
↓
state + audit history + governance
↓
local JSON package
```

This improves the manual participation loop.

The character can now be inspected, adjusted, governed, and exported with its human intervention history attached.
