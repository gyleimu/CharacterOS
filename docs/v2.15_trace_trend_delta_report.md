# CharacterOS V2.15 Trace Trend Delta 阶段报告

## 版本结论

当前项目进入：

```text
CharacterOS V2.15 Trace Trend Delta
```

这一阶段继续完善 V2 Personality Galaxy replay summary。V2.15 不新增世界观、不做 3D，也不进入 V3，而是让 replay summary 更直接地表达“趋势变化量”。

## 本阶段新增

### 1. Force Delta

更新：

```text
src/core/trace/traceReplaySummary.ts
```

`TraceReplaySummary` 新增：

```text
forceDelta
```

含义：

```text
lastForce - firstForce
```

当前只聚焦 V2 最关键的调试维度：

```text
trust
fear
```

### 2. Velocity Delta

新增：

```text
velocityDelta
```

含义：

```text
lastVelocity - firstVelocity
```

用于直接观察人格惯性是否在长序列中持续累积。

### 3. Dominant Cluster Category

新增：

```text
dominantClusterCategory
```

计算方式：

```text
选择 massDelta 最大的 cluster trend
```

用于快速判断当前 replay 中真正主导人格牵引的是哪个星团。

## 测试更新

## 当前意义

Replay summary 现在可以直接回答：

```text
这条心理轨道最终往哪里漂？
牵引力是否增强？
人格速度是否累积？
哪个星团是主导星团？
```

这让 summary 更适合作为未来 Dashboard 表格、实验比较和 Benchmark seed 的输入。
