# CharacterOS V10.49 — Mind Galaxy Debug Instrument Controls Report

**Date:** 2026-06-28
**Status:** Complete
**Scope:** Static preview/artifact observation controls only.

## Summary

V10.49 adds a small, collapsed-by-default debug instrument panel to the Mind Galaxy preview and static artifact.

This is not a product dashboard and not an editor. It is a developer observation aid for reproducing a visible frame, freezing deterministic drift time, and capturing compact snapshot summaries.

## Added Controls

| Control | Purpose | State Mutation |
|---|---|---|
| Debug | Opens a compact instrument panel | None |
| Freeze Time | Switches drift sampling from `performance.now()` to fixed `timeMs` | None |
| Step +1s | Advances fixed debug time by 1000 ms | None |
| Step +10s | Advances fixed debug time by 10000 ms | None |
| Reset Time | Sets fixed debug time to 0 and freezes time | None |
| Outer Nebula | L0, subtle drift, motion on | View only |
| Memory Field | L2, subtle drift, motion on | View only |
| Drift Inspect | L4, visible drift, motion on, running time | View only |
| Copy Snapshot Summary | Writes a compact summary to clipboard or the debug textarea | None |

## Debug Panel Fields

- scale
- zoom level
- data source
- motion state
- drift mode
- current timeMs
- visible node count
- visible edge count
- real data status

## Important Fix

The artifact export previously embedded `window.__MIND_GALAXY_PREVIEW_DATA__` after the external preview script.

That meant file-based artifact opening could still miss embedded data because the preview script executed first. V10.49 now injects embedded data before `mind-galaxy-preview.js`, making the artifact genuinely self-contained for `file://` usage.
