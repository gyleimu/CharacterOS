## Architecture Meaning

CharacterOS now has a fully traceable manual calibration loop:

```text
observe
↓
propose
↓
preview
↓
audit
↓
apply
↓
rollback
↓
history
```

This matters because a character's parameters should never silently change.

Every manual change should leave a trail.
