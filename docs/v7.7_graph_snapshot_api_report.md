## 1. 新增文件

### `src/app/api/characters/[characterId]/graph/route.ts`（75 行）

**GET /api/characters/[characterId]/graph**

- 只读 — 不修改任何状态
- 不需要 API key
- 不调用 LLM
- 从 character state 构建 MindGraphSnapshot

**可选 query params：**

| Param | 效果 |
|-------|------|
| `?includeISF=true` | 包含 internal_state_variable 节点（来自 InternalStateField） |
| `?includeBenchmark=true` | 运行 V6 benchmarks，包含 benchmark_signal 节点 |

### `src/appContracts/characterPhysics.ts`（新增类型）

```typescript
export interface GetGraphSnapshotResponse {
  snapshot: MindGraphSnapshot;
}
```
