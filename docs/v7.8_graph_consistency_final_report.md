## 2. Graph Builder Consistency Audit

| Check | Status | Evidence |
|-------|--------|----------|
| Pure projection — no state mutation | ✓ | `buildMindGraphSnapshot` never writes to state. Test: `does not mutate the input state`. |
| No LLM calls | ✓ | No import of any LLM module. |
| No external mutation | ✓ | Pure functions only. `deriveNeedDeficiencies`, `deriveDesires`, `deriveBehaviorBiases` are all pure. |
| Deterministic output | ✓ | Test: `produces deterministic output for same state`. Same state → identical snapshot. |
| generatedAt injectable | ✓ | Options support `generatedAt` override for test determinism. |
| Validator integrated at build time | ✓ | `validateMindGraphSnapshot` called before return. Violations injected as warnings. |
| Warnings never throw | ✓ | All validation issues collected into `warnings[]`. No runtime throws. |
| options.include semantics clear | ✓ | `minNeedIntensity`, `minBeliefStrength`, `minDesireIntensity`, `minBiasLikelihood` all documented. |
| ISF nodes optional | ✓ | Only added when `options.internalStateField` is provided. |
| Benchmark signal optional | ✓ | Only added when `options.benchmarkResults` is provided. |

**Audit result: PASS.** No issues found. Builder is a pure projection with no side effects.
