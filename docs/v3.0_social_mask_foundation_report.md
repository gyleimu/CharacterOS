# CharacterOS V3.0 Social Mask Foundation Report

## Version

```text
CharacterOS V3.0 Social Mask Foundation
```

## Goal

V3.0 introduces the first Social Mask and Multi-State model.

The goal is to represent that humans do not always say what they feel, and do not always do what they say.

## Core Model

```text
True State
-> Conscious State
-> Expressed State
-> Behavior State
```

These states may conflict.

## New Core Module


## Integration

`DerivedCharacterState` now includes:

```text
socialExpression
```

Social Mask V0 uses:

```text
PersonalityCoordinate
MetaState
PsychologicalBoundary
DesireState
BehaviorBias
BehaviorDecision
EmbodiedAction
```

It does not modify the decision.

It explains the mismatch between:

```text
what the character wants
what the character tells himself
what the character says
what the character actually does
```

## Meaning

CharacterOS now models a crucial human inconsistency:

```text
true feeling != conscious explanation != spoken words != final behavior
```

This is the first step toward characters who can hide, soften, deny, rationalize and protect themselves socially.
