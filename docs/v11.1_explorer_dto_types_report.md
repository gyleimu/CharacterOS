## 5. Safety Boundary

- All DTOs include safety notes or disclaimers
- `CharacterStateSurface.safetyNote` — visible alongside state
- `RealityAuditPanel.disclaimers` — co-located with verdict
- `MindGalaxyEmbed.safetyBoundary` — research view only warning
- `ExplorerManifest.safetyDisclaimers` — platform-level disclaimers
- No DTO exposes raw numeric coordinate values
- All personality values use qualitative bands (low/moderate/high)
- All emotional labels use Chinese qualitative descriptions

## 6. Why No UI / API (Yet)

V11.1 is a **data contract milestone**, not a feature milestone.

- Types and builders establish the stable interface between V10 core and future V11 UI
- Tests verify: determinism, no mutation, safety boundaries, qualitative output
- Next: V11.2 Event Studio Preview Core builds on these DTOs
