# CharacterOS
a new ideal in a normal night 

CharacterOS

An experiment on experience-driven personality evolution.

Core Ideas

Most character systems define personalities through fixed traits.

I wonder whether personality should emerge from experiences instead.

Experiences
↓
Beliefs
↓
Deficiencies
↓
Desires
↓
Decisions
↓
New Experiences

Some thoughts:

- Characters are defined by experiences, not traits.
- Desires emerge from deficiencies.
- Memory behaves more like a space than a list.
- Similar memories form clusters.
- Frequently recalled memories move toward the core.
- Personality evolves over time.
- Characters should drive the story instead of serving the plot.

Maybe personality is not a set of traits.

Maybe personality is an evolving memory galaxy.

This repository is simply a place to record ideas and experiments.

No grand claims.

Just curiosity.
