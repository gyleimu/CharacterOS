## 1. 新增文件

### `src/core/temporal/subProcessTrace.ts`（177 行）

**类型定义：**

| 类型 | 用途 |
|------|------|
| `TemporalSubProcessKind` | 4 种 subprocess 的判别联合：`memory_decay \| procedural_decay \| boundary_recovery \| reward_recovery` |
| `TemporalSubProcessTrace` | 基础接口（id, kind, label, reads, writes, changedStates, reasons, metrics） |
| `MemoryDecaySubProcessTrace` | memory decay 专用：5 个 metrics（count, recency before/after, weight before/after） |
| `ProceduralDecaySubProcessTrace` | procedural decay 专用：3 个 metrics（count, strength before/after） |
| `BoundaryRecoverySubProcessTrace` | boundary recovery 专用：6 个 metrics（stress/integrity/cracks before/after） |
| `RewardRecoverySubProcessTrace` | reward recovery 专用：6 个 metrics（dopamine/threshold/craving before/after） |
| `AnySubProcessTrace` | 联合类型 |
| `SubProcessTraceSummary` | 跨 subprocess 汇总（total, kinds, changedStateNames, warnings, reasons） |

**Pure Helper：**

```typescript
summarizeSubProcessTraces(traces) → SubProcessTraceSummary
```

不读不写 CharacterPhysicsState。不修改输入数组。对空输入产生 warning。
