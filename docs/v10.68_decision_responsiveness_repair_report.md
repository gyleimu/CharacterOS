## What The Decision Layer Now Consumes

Examples of structural mappings:

- trust down -> caution/testing/withdrawal up, openness down
- fear up -> caution/freeze/withdrawal up
- attachment up -> reassuranceSeeking/boundaryProbing up
- boundary stress up -> withdrawal/freeze/boundary protection up
- supportive memory / relief -> openness/repair up, withdrawal/caution softened
- need pressure up -> cautious and evidence-seeking strategies gain weight

Scenario relevance is part of the calculation, so the same state delta does not blindly affect every follow-up scenario.
