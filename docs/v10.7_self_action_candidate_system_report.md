## Objective

Establish the self-action candidate system — the layer where the character internally tends toward doing something, without evaluating, choosing, or executing the action. This is pre-behavior, not behavior.

---

## Why Candidates Are Not Actions

A self-action candidate means:

> The character internally tends toward doing something, but has not yet decided to do it.

In CharacterOS, the gap between "I want to sleep" and actual sleep is:
- Evaluation (is this appropriate right now?)
- Selection (which action among several?)
- Execution (actually doing it)

V10.7 only generates the first layer — the **tendency** — with full traceability. Each candidate has:
- **Strength** — how strongly the drive pushes
- **Urgency** — how soon it wants to happen
- **Friction** — internal resistance to acting
- **Risk** — potential downside
- **Score** — composite = strength × 0.5 − friction × 0.3 + urgency × 0.2

### Scoring Rules (9 action types)

| Type | Primary Drivers | Friction Sources |
|---|---|---|
| `sleep` | sleepPressure 40%, fatigue 35% | restlessness, inspiration |
| `do_nothing` | low energy 30%, fatigue 25% | selfControl, restlessness |
| `write_note` | inspiration, question/self_talk thought | fatigue, irritability |
| `go_for_walk` | restlessness 35%, exploration 35% | fatigue, sleep phase |
| `check_phone` | loneliness 30%, restlessness 20% | selfControl |
| `avoid_message` | stress 35%, irritability 30% | (low selfControl amplifies) |
| `revisit_memory` | boredom 30%, loneliness 25% | — |
| `seek_contact` | loneliness 40%, desire 30% | stress, fatigue, selfControl |
| `withdraw` | stress 35%, irritability 35% | — |

**Sleep suppression:** All non-sleep/non-do_nothing actions receive +0.5 friction during sleep phases.

**Score formula:** `clamp(strength × 0.5 − friction × 0.3 + urgency × 0.2)`

## Design Philosophy

> A candidate is not an action. It is a pre-action, a tendency, an internal leaning.

Before a character *does* something, they must first *tend toward* doing it. V10.7 models that tendency layer — with all the internal conflicts (drive vs friction, urgency vs risk) that shape which actions surface as candidates.

The character may want to sleep (high fatigue) but also wants to go for a walk (high restlessness). Both appear as candidates. Neither executes.

> First the leaning. Then the choice.
