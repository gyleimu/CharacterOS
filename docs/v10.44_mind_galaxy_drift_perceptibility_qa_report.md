## 1. Executive Summary

V10.44 addresses user feedback: "looks like nothing is moving." The drift animation was too subtle to perceive even at L4. This phase adds a **dual-mode drift system**: `subtle` (default, kept for product feel) and `visible` (for validation/observation). Also adds faint orbit ghost hints at L4 so users can see where nodes are anchored.

**No core changes. No API. No state mutation. Preview-only.**

## 2. Drift Mode Design

### Subtle Mode (default)

| Parameter | Value | Purpose |
|-----------|-------|---------|
| Period | 55–110 seconds | Very slow — barely perceptible |
| Amplitude | 0.5–2 screen px | < node radius, psychological field feel |
| Orbit ghosts | Visible at opacity 0.12 | Faint anchor hint |

### Visible Mode (toggle)

| Parameter | Value | Purpose |
|-----------|-------|---------|
| Period | 25–50 seconds | 2× faster — clearly perceptible |
| Amplitude | 1.5–6 screen px | 2.5× larger — visible but stays < node radius |
| Orbit ghosts | Visible at opacity 0.12 | Same anchor hint |

### Gating

| Condition | Drift active? | Orbit ghosts? |
|-----------|--------------|---------------|
| L0–L3 (any mode) | No | No |
| L4 + subtle | Yes, very slow | Yes |
| L4 + visible | Yes, faster | Yes |
| Paused | No (frozen) | No |
| Reduced Motion | No | No |

## 3. Orbit Ghost Hints

At L4, each node renders a **faint circle at its original anchor position** (zIndex: node.z - 1). This is drawn *before* the drifted node, so the drifted node overlaps it. Users can see:

- A ghost ring where the node "belongs"
- The solid node drifted slightly from that position
- Drift vectors pointing in the direction of motion

Ghosts are rendered with:
- `fill: "transparent"`
- `stroke`: same as node's stroke color
- `strokeWidth: 0.5`
- `opacity: 0.12 × driftVis × lodOpacity`

## 4. HUD Changes

```
Before: scale 6.50 · L4 · running · motion:normal · 13 nodes · fixture · real:loading
After:  scale 6.50 · L4 · running · motion:normal · 13 nodes · fixture · real:loading · drift:visible
```

Drift display:
- L0–L3: `drift:—` (dimmed, no drift active)
- L4 + subtle: `drift:subtle` (green anim class)
- L4 + visible: `drift:visible` (green anim class)
- Reduced Motion: `drift:—` (regardless of zoom)
