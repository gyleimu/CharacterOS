# CharacterOS V2.25 Procedural Memory Foundation Report

## Version

```text
CharacterOS V2.25 Procedural Memory Foundation
```

## Goal

V2.25 adds the first low-level habit layer.

The goal is to represent actions that do not require reflection:

```text
cue -> automatic routine -> action
```

This supports the Architecture Bible principle:

```text
Humans do not constantly think.
Humans constantly live.
Most of life is instinct and habits.
```

## New Core Module

```text
src/core/procedural/proceduralMemory.ts
```

New types:

```text
ProceduralRoutine
ProceduralCue
ProceduralActivation
```

New functions:

```text
activateProceduralMemory()
reinforceProceduralRoutine()
decayProceduralRoutine()
```

## Model

Each routine contains:

```text
id
cueTags
action
strength
repetitionCount
lastTriggeredAt
```

Activation uses:

```text
cueMatch
routine strength
repetition count
MetaState.attention
MetaState.selfControl
PsychologicalBoundary stress ratio
```

It outputs:

```text
automaticity
activationScore
reasons
```

## Integration

`CharacterPhysicsState` now contains:

```text
proceduralRoutines
```

`SerializedCharacterPhysicsState` also preserves procedural routines.

`runContinuousTick()` now decays unused procedural routines and returns:

```text
proceduralRoutineCount
averageProceduralStrengthBefore
averageProceduralStrengthAfter
```

Coverage:

```text
high pressure and low self-control increase automaticity

```

## Current Boundary

V2.25 does not implement:

```text
habit conflict resolution
motor skill learning
background scheduler
relationship-specific routines
LLM-generated routines
```

Procedural Memory V0 is deterministic and pure logic. It is ready to become part of simulation, Dashboard and future continuous living loops.

## Meaning

CharacterOS now has three different action layers:

```text
BehaviorDecision      -> reflective psychological decision
ActionNoise           -> imperfect embodied execution
ProceduralMemory      -> automatic low-level habit
```

This moves CharacterOS further away from a prompt-only character and closer to a layered living mind.
