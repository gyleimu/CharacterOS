## Objective

Expand the boredom system beyond a single scalar into a multi-dimensional psychological model, and establish the minimum inspiration seed system — generating candidate signals for creative insight without executing insight events.

Boredom is not emptiness. It drives attention drift, restlessness, daydreaming, exploration pressure, and creative incubation.

---

## Why Boredom Is Not a Failure State

Most simulation systems treat boredom as "nothing is happening" — a state to escape. In CharacterOS, boredom is **productive psychological space**:

- **Restlessness** — physical/mental urge to change state
- **Daydreaming** — mind wanders, associations form
- **Exploration pressure** — desire to seek new stimuli or perspectives
- **Irritability** — frustration when boredom persists under stress/fatigue

A character should not always be事件-filled. They should also get bored, drift, daydream, and have random associations — because humans do.

## Architecture

### BoredomExpansionState (5 dimensions)

| Field | [0,1] | Description |
|---|---|---|
| `boredom` | 0.25 | Core boredom level |
| `restlessness` | 0.20 | Physical/mental urge to act |
| `daydreamingTendency` | 0.35 | Mind-wandering propensity |
| `explorationPressure` | 0.25 | Drive to seek novel stimuli |
| `irritability` | 0.15 | Frustration from sustained boredom |

### BoredomExpansionContext

| Field | Source | Role |
|---|---|---|
| `stimulationLevel` | Input (external) | Low → boredom grows |
| `socialContactLevel` | Input (external) | Low → boredom grows |
| `energy` | EnergyFatigueState | Must have energy to feel bored |
| `fatigue` | EnergyFatigueState | High fatigue suppresses boredom |
| `curiosity` | MetaState | Amplifies exploration |
| `stressLoad` | Boundary | Amplifies restlessness/irritability |
| `sleepQuality` | SleepWakeState | Boosts daydreaming |

### Delta Rules

**Boredom** grows when:
- `stimulationLevel` is low → stimulation deficit × 0.8
- `socialContactLevel` is low → social deficit × 0.5
- Gated by energy (>0.1 required) and fatigue suppression
- Existing boredom provides inertia (dampening)

**Restlessness** grows when:
- Boredom high (0.4× link) × energy sufficient (0.6× gate)
- Stress > 0.5 adds boost
- High stimulation (>0.6) provides mild decay

**Daydreaming** increases when:
- Boredom in optimal range (0.2–0.7)
- Sleep quality boosts it (+25%)
- High stress (>0.7) blocks it
- High restlessness penalizes it

**Exploration Pressure** increases when:
- Boredom × Curiosity (0.4× boost)
- Very high fatigue (>0.8) gates it

**Irritability** increases when:
- Boredom combined with fatigue (0.5×) and stress (0.35×)

All deltas use `timeFactor = min(elapsedHours / 24, 1)`.

## Inspiration Seed System

### InspirationSeedType (5 types)

```
memory_association | problem_reframing | creative_image |
social_realization | quiet_realization
```

### Generation Rules

| Condition | Seed Type | Source |
|---|---|---|
| Daydreaming > 0.3 | `creative_image` / `memory_association` | `daydreaming` |
| Exploration > 0.3 + Curiosity > 0.4 | `problem_reframing` | `boredom` |
| Dream fragment present + generated | `creative_image` / `quiet_realization` | `dream_residue` |
| Memory count > 3 + Boredom > 0.2 | `memory_association` | `memory_resurfacing` |
| Boredom > 0.45 + Irritability < 0.5 | `quiet_realization` | `boredom` |

All candidates have `evaluated: false` — seeds only, not insight events.

Irritability reduces probabilities across all seed types (creativity suffers when frustrated).

## Design Philosophy

> Boredom is not a failure state. Boredom is the blank space of a life system.

Humans drift when bored, get restless, daydream, associate loosely, get irritable — and sometimes, something new emerges from the empty space. An insight. A connection. A quiet realization.

CharacterOS should not fill every moment with events. Characters should also be able to:
- Be bored
- Drift inattention
- Think about nothing important
- Suddenly have an idea

But V10.5 only generates **inspiration seeds**. The actual insight event — what the character does with the seed — comes later.
