# CharacterOS V2.9 Cluster Metrics Guard 阶段报告

## 版本结论

当前项目进入：

```text
CharacterOS V2.9 Cluster Metrics Guard
```

这一阶段继续收敛 V2 Personality Galaxy 的星团指标逻辑，重点不是增加新功能，而是让 cluster mass / density / stability 在边界条件下更可信。

## 本阶段修正

### 1. 空星团不再凭空稳定

更新：

```text
src/core/galaxy/clusterMetrics.ts
```

此前当一个 `ImpactCluster` 没有挂载任何 `MemoryNode` 时，`variance = 0` 会让 `stability = 1`。

这在物理语义上不合理：

```text
没有记忆支撑的星团
不应该自动成为稳定星团
```

现在的逻辑是：

```text
如果没有 cluster memories：
  保留 cluster.mass
  保留 cluster.density
  保留 cluster.stability
  variance = 0
```

### 2. Density / Stability 边界测试

更新：

```text
tests/core/personalityGalaxy.test.ts
```

新增测试：

```text
空星团保留已有 mass / density / stability
集中记忆拥有更低 variance
集中记忆拥有更高 density
集中记忆拥有更高 stability
集中记忆和离散记忆在 importance 相同时 mass 相同
```

## 当前意义

V2 的星团指标现在更接近 Personality Galaxy 的原始设定：

```text
mass 来自重要性和重复次数
density 来自同类经历是否集中
stability 来自经历是否稳定聚合
```

这为后续 replay 长序列、Benchmark seed 和更复杂的 cluster variance 分析打下了更稳的基础。
