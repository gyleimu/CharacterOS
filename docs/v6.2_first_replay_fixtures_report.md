## 1. 新增文件

### `src/core/benchmark/fixtures/firstReplayFixtures.ts`（335 行）

**5 个 benchmark case 定义 + 3 个 helper functions。**

#### 5 个 Fixtures

| # | id | category | events | ticks | directions | 覆盖能力 |
|---|-----|----------|--------|-------|------------|---------|
| A | `abandonment_event_lowers_trust` | `event_impact` | 1 abandonment | 1 tick (7d) | trust↓, fear↑, impact billed | event → coordinate pipeline |
| B | `waiting_event_raises_fear` | `event_impact` | 1 silence (3d) | 1 tick (3d) | fear↑, neuroticism bounded | 短期 anxiety 通路 |
| C | `memory_should_decay_over_time` | `memory_decay` | 1 abandonment | 2 ticks (7d, 30d) | recency↓, weight↓, count bounded, subprocess monotonic | V5 subprocess metrics |
| D | `boundary_should_recover_under_rest` | `homeostasis_recovery` | 2 events (abandonment + betrayal) | 1 tick (30d) | stress↓, integrity bounded, cracks bounded | V5 boundary_recovery subprocess (D10-aware) |
| E | `belief_should_strengthen_with_repeated_evidence` | `belief_evolution` | 3 abandonment (repeated) | 2 ticks (7d, 14d) | belief strength↑, belief array non-empty | V4 belief evolution (delegated phase) |

#### 关键设计决定

```text
1. 全部使用 directional/default tolerance — 不追求精确数值
2. 每个 expected direction 附带 human-readable reason
3. reusable event templates 用 const 定义 — 可复用、可维护
4. initialState 使用 { kind: "default", characterName: "lin_fan" }
   — 不内联 state object（fixture 不绑定特定 state snapshot）
5. 多 tick fixture（C, E）使用 monotonic tolerance 验证连续方向
6. D10-aware: fixture D 的 metrics 指向 boundary_recovery subprocess
```
