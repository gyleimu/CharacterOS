## 2. 这是 Instrumentation，不是 Delegation

```text
V4.7-V4.12 的 temporal adapters 是 delegation：
  - 将 mutation logic 从 runContinuousTick 提取到独立的 adapter 函数
  - runContinuousTick 通过 adapter 调用，不再直接执行 V3 函数

V5.2 memory decay subprocess 是 instrumentation：
  - decayMemory() 调用仍在 runContinuousTick Phase 3 中 inline 执行
  - buildMemoryDecaySubProcessTrace() 是纯 trace builder，只读不写
  - 不改变任何 mutation 的位置、顺序或输入/输出
```

| 属性 | Delegation (V4) | Instrumentation (V5.2) |
|------|-----------------|------------------------|
| mutation 位置 | 移到 adapter | 保持在 runContinuousTick |
| 调用方式 | 通过 adapter | 直接调用 V3 函数 |
| trace 来源 | adapter 内部 | 纯 builder 函数 |
| 行为影响 | 必须保持等价（D1-D10） | 零影响 |
| trace 字段 | phase trace | subProcess trace（附加） |
