## Design Constraint

Adjustment drafts are not commands.

They are review artifacts.

CharacterOS still refuses to automatically rewrite baselines from one observation chain.

## Architecture Meaning

This gives the project a safer manual calibration workflow:

```text
observe
accumulate
cross threshold
generate draft
human review
```

The system is now closer to the user's requested state:

The operator can participate in data adjustment, but CharacterOS still protects personality continuity and avoids instant mutation.
