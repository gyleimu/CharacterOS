## 1. 新增文件

### `src/core/graph/mindGraphValidator.ts`（300 行）

**15 条结构完整性检查 + node-specific risk projection.**

| 检查 | Code | Severity |
|------|------|----------|
| Duplicate node id | V001 | error |
| Duplicate edge id | V002 | error |
| Missing source node | V003 | error |
| Missing target node | V004 | error |
| Node weight out of [0,1] | V005 | error |
| Edge weight out of [0,1] | V006 | error |
| Node mass not finite | V007 | error |
| Invalid node type | V008 | error |
| Invalid edge type | V009 | error |
| Invalid node source | V010 | error |
| Invalid edge evidence | V011 | error |
| Self-loop (not allowed) | V012 | error |
| Summary node count mismatch | V013 | error |
| Summary edge count mismatch | V014 | error |
| Orphan node (non-core) | V015 | warning |
| Risk without mass/weight | V016 | warning |
| Confidence without weight | V017 | warning |

**`projectNodeRisk(node)` — 5 种节点类型的 risk projection：**

| Node type | Heuristic | Thresholds |
|-----------|-----------|------------|
| impact_cluster | cluster.mass | high ≥0.6, medium ≥0.3 |
| memory | recency | medium ≥0.7 |
| belief | strength | high ≥0.6, medium ≥0.3 |
| need | intensity | high ≥0.6, medium ≥0.35 |
| personality_core | stress = (1-trust)*0.5 + fear*0.5 | high ≥0.7, medium ≥0.4 |

### `src/core/graph/mindGraphProjection.ts`（130 行）

**Edge rationales — 解释每条边为什么存在.**

| 函数 | 用途 |
|------|------|
| `explainMindGraphEdges(edges, nodeLabelMap)` | 为每条边生成人类可读的解释 |
| `buildNodeLabelMap(nodes)` | 从节点数组构建 id→label map |

每种 edge type 有专门的解释模板，包括 weight、contribution、formula 引用。

### `src/core/graph/mindGraphBuilder.ts`（修改）

- 集成了 `validateMindGraphSnapshot` — builder 返回前自动验证
- 集成了 `projectNodeRisk` — 节点自动 risk projection
- 验证失败以 `[severity:code]` 格式注入 warnings
