```text

rollback count
blocked count
total operations
unique target paths
latest target paths
frequent target paths
stability risk
reasons
```

## Architecture Meaning

Manual calibration now has governance signals.

CharacterOS can answer:

```text
How often has this character been manually changed?
Which parameters are repeatedly touched?
How often are changes rolled back?
Is manual calibration becoming unstable?
```

This keeps the project aligned with the Parameter Evolution principle:

Values are not the core.

The evolution and governance of values are the core.
