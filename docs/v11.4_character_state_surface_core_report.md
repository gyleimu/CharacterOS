## 2. From Raw State to Human-Readable Surface

```
CharacterPhysicsState (raw)
    │
    ├─ deriveNeedDeficiencies() ────► Needs (by intensity)
    ├─ deriveDesires()        ────► Goals (by urgency)
    ├─ beliefStates           ────► Beliefs (by strength, sorted)
    ├─ coordinate.values      ────► Personality bands (qualitative)
    ├─ boundary.phase         ────► Stress label
    ├─ recent events/audit    ────► Headline + emotional context
    │
    └─ CharacterStateSurface (human-readable)
```

## 3. Surface Sections

| Section | Source | Output |
|---------|--------|--------|
| **Headline** | boundary + fear + recent events | 一句话状态摘要 |
| **Emotional State** | fear, trust, neuroticism, recent context | primary/valence/arousal + label |
| **Stress State** | boundary phase, stressLoad/capacity | level/phase + risk label |
| **Dominant Needs** | deriveNeedDeficiencies → top 3 | name/intensity |
| **Dominant Beliefs** | beliefStates → sorted by strength → top 3 | content/strength |
| **Active Goals** | deriveDesires → top 3 | content/urgency |
| **Behavior Tendencies** | fear, trust, openness, safety need | action/strategy/caution/openness |
| **Personality Summary** | coordinate → qualitative bands | trust/fear/openness/attachment/neuroticism |
| **Safety Note** | hardcoded | simulation-not-diagnosis |
| **Source Snapshot** | fingerprint hash | snap_{hex} |

## 4. What Is Intentionally Hidden

- ❌ Raw coordinate values (0.2647 → "信任偏低")
- ❌ Internal state IDs (particleIds, clusterIds)
- ❌ Engine internals (driftMultiplier, velocity)
- ❌ Biological nature
- ❌ Reward state, homeostasis state, boredom state
- ❌ Procedural routines

## 5. Banding Rules

| Raw Range | Band |
|-----------|------|
| > 0.6 | high |
| 0.3–0.6 | moderate |
| < 0.3 | low |

Qualitative descriptions use 4 levels for personality labels.

## 6. Headline Logic

```
overflow → "心理压力超载，急需恢复"
strained + high fear → "关系压力升高，正在寻求确认"
strained → "压力偏高，但仍在管理范围内"
low fear + high trust → "情绪稳定，信任感较高"
high neuroticism → "情绪波动较大"
recent support → "正向支持事件后，缓慢恢复中"
recent abandonment → "近期负面事件影响仍在持续"
default → "当前状态相对稳定"
```

## 7. Safety Boundary

- Safety note visible on every surface: "模拟输出，非医学/心理诊断"
- No raw numbers exposed as clinical indicators
- All labels are qualitative Chinese descriptions, not diagnostic terms
- Source snapshot enables audit trail
