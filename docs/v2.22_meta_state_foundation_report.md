## 测试覆盖

更新：

```text
tests/core/continuousTick.test.ts
```

新增验证：

```text
高 memoryStrength / 低 forgettingSpeed 会降低有效遗忘速度
低 memoryStrength / 高 forgettingSpeed 会提高有效遗忘速度
高压力会提高 emotionalSensitivity
高压力会提高 traumaAmplification
高压力会降低 resilience
高压力会降低 selfControl
```

## 当前意义

CharacterOS 从：

```text
状态会变
```

进一步推进到：

```text
改变状态的参数也会变
```

这让人物不只是被经历改变，还会改变自己“如何被经历改变”。
