# CharacterOS V2.23 Attention System Foundation 阶段报告

## 版本结论

当前项目进入：

```text
CharacterOS V2.23 Attention System Foundation
```

这一阶段继续落实 Architecture Update V2：人物不是客观接收世界，而是被当前 meta state、情绪压力和心理边界过滤后，选择性地注意世界。

## 本阶段新增

### 1. Attention System V0

新增：

```text
src/core/attention/attentionSystem.ts
```

核心类型：

```text
AttentionChannel
AttentionProfile
AttentionEvaluation
```

当前 channel：

```text
danger
relationship
reward
novelty
control
```

### 2. Attention Profile

新增：

```text
buildAttentionProfile({ meta, boundary })
```

根据：

```text
MetaState
PsychologicalBoundary
```

计算人物当前注意力偏置。

### 3. Event Attention Evaluation

新增：

```text
evaluateEventAttention({ event, meta, boundary })
```

输出：

```text
profile
dominantChannel
noticedTags
score
reasons
```

例如：

```text
高 traumaAmplification + 高 emotionalSensitivity -> 更关注 danger
低 lonelinessTolerance + 高 attachmentStyle -> 更关注 relationship
```

### 4. Continuous Tick Trace

更新：

```text
src/core/time/continuousTick.ts
```

Tick trace 新增：

```text
attentionProfileBefore
attentionProfileAfter
```

这让连续生活系统可以观察人物的注意力偏置如何随 meta state 和 boundary 变化。

### 5. Architecture Bible Update

更新：

```text
docs/architecture_bible.md
```

新增：

```text
Attention System
```

## 当前意义

CharacterOS 现在开始表达：

```text
同一个世界
不同的人会注意到不同的东西
同一个人
在不同心理状态下也会注意到不同的东西
```

这为后续 Context Loading、Deep Thinking 触发和行为噪声打基础。
