## 本阶段新增

### 1. Replay Artifact Version

更新：

```text

```

新增：

```text
TRACE_REPLAY_SCHEMA_VERSION = "characteros.trace-replay.v1"
```

`TraceReplayArtifact` 现在包含：

```text
schemaVersion
createdBy
parameters
```

### 2. Scenario Metadata

`TraceReplayScenarioDefinition` 现在包含：

```text
id
title
description
expectedForces
buildEvents
```

Artifact 中新增：

```text
scenarioMeta.title
scenarioMeta.description
scenarioMeta.expectedForces
```

这让 replay JSON 不只是数据，还携带实验意图。

### 3. Replay Builder 稳定化

`buildTraceReplayArtifact` 现在将以下参数写入 artifact：

```text
daysPerStep
learningRate
```

后续比较不同 replay 输出时，可以知道模拟参数。
