# CharacterOS V3.3 Meaning System Foundation Report

## Version

```text
CharacterOS V3.3 Meaning System Foundation
```

## Goal

V3.3 introduces the first Meaning System.

The goal is to separate:

```text
need
reward
meaning
```

These are related, but not the same.

Meaning may override reward.

Meaning may override pain.

CharacterOS can now represent:

```text
what the character needs
what the character wants
what rewards the character chases
```
