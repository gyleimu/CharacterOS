# CharacterOS V3.7.20 Calibration Hint List Refactor Report

## Summary

This update extracts shared calibration-hint rendering into a reusable UI component.

```text
src/components/character/CalibrationHintList.tsx
```

## Updated Components


## Why

Parameter Network and Homeostasis now both produce calibration hints.

Future systems such as Recovery, Reward, Boundary, and Benchmark may also produce hints.

Using one shared renderer avoids repeating:

```text
severity styling
hint card structure
value formatting
```
