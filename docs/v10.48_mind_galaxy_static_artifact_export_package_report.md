## 4. Key Design Decisions

### Embedded Data
The `index.html` injects `window.__MIND_GALAXY_PREVIEW_DATA__` as an inline `<script>` tag containing the full `MindGalaxyPreviewData` JSON. This bypasses `file://` CORS restrictions that block `fetch()` to local files.

### Dual Data Path
The preview JS now checks two sources:
1. `window.__MIND_GALAXY_PREVIEW_DATA__` (artifact mode) — always works
2. `fetch("mind-galaxy-real-data.json")` (local server mode) — works with HTTP server

### Contract-Validated
Every export runs `validateMindGalaxyPreviewData()` before writing. Artifact data is guaranteed contract-compliant.
