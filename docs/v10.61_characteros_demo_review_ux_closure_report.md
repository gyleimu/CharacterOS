# V10.61 CharacterOS Demo Review UX Closure Report

**Date:** 2026-06-28
**Version:** 10.61.0
**Previous:** 10.60.0 (Scenario Mini Flow + Intensity + Density QA)

## Overview

V10.61 closes four review UX gaps: tab state survives refresh, Galaxy iframe defers load, scenario action text is expandable, and the demo prints cleanly for markup/review.

## Changes

### 1. Tab Hash Persistence

Tab selection persists across page refresh using `location.hash`.

**Implementation details:**
- `KNOWN_TABS` array validates: `overview`, `today`, `decision`, `scenarios`, `life`, `galaxy`
- Tab click handler calls `history.replaceState(null, "", "#" + tab)` to update URL without creating history entries
- `hashToTab()` parses `location.hash`, returns `null` for invalid values
- `restoreTabFromHash()` activates the tab if hash is valid; overwritten hash fallback stays on Overview
- `hashchange` event listener catches browser back/forward
- **No localStorage, no sessionStorage, no cookie** — pure URL-based, read-only

### 2. Galaxy Iframe Lazy Loading

The Mind Galaxy iframe no longer loads on page open.

**Implementation details:**
- HTML iframe has `data-src` attribute instead of `src`
- `galaxyLoaded` flag prevents duplicate loading
- `loadGalaxyIframe()` checks `galaxyLoaded`, sets `iframe.src = data-src` on first call
- Called from `activate("galaxy")` — triggers when user clicks Galaxy tab
- "全屏打开" link preserved (always available, opens in new tab)
- Once loaded, stays loaded across tab switches

### 3. Action Expand/Collapse Toggle

Scenario mini-flow action text now has an expand/collapse toggle.

**Implementation details:**
- `scenarioMiniFlow(s, idx)` generates unique ID `sact-{idx}` per scenario
- Only scenarios with action >28 chars get a toggle button
- Button data attributes: `data-full` (complete text), `data-short` (truncated text)
- Click toggles between "展开完整行动" and "收起"
- Pure DOM mutation via `textContent` swap — no state, no storage, no API
- `.action-toggle` styled as a subtle inline button

### 4. Print Stylesheet

`@media print` block added to CSS for clean printed review copies.

**Implementation details:**
- White background (`#fff`), black text (`#111`) for readability
- Tab navigation hidden (`display:none`)
- All `.panel` forced visible (`display:block!important`)
- Dark-card backgrounds reset to `#fafafa` or `#fff`
- Borders and accents softened to grayscale
- Iframe hidden (can't print interactive content)
- All toggle buttons and "全屏打开" links hidden
- Scenario grid collapses to single column for paper width
- Pages break avoided within panels (`page-break-inside: avoid`)
- Tables, cards, flow nodes, mini-flows all render in grayscale
