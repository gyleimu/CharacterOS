## 3. Boredom Delegation 模板提炼

以下模式可直接复用于 V4.8 belief_evolution delegation：

### Adapter 结构模板

```typescript
// 1. 定义与 V3 函数一致的输入类型
export interface XxxAdapterInput { /* mirrored from V3 function params */ }

// 2. 定义返回类型（V3 trace + phase metadata + temporalResult）
export interface XxxAdapterResult {
  trace: V3TraceType;           // 直接委托给 V3 函数的返回值
  phase: { name, changedStates, reasons };
  temporalResult: TemporalProcessResult;
}

// 3. 执行函数：委托给 V3，返回等价结果
export function executeXxxAdapter(input: XxxAdapterInput): XxxAdapterResult {
  const trace = v3Function(input);
  return { trace, phase: { name, changedStates, reasons }, temporalResult: { ... } };
}
```

### runContinuousTick 接入模板

```typescript
// 之前（inline）
const result = v3Function({ field: state.field, ... });
state.field = result.after;
phases.push({ name: "phase_name", changedStates: ["field"], reasons: result.reasons });

// 之后（delegated）
const adapterResult = executeXxxAdapter({ field: state.field, ... });
state.field = adapterResult.trace.after;
phases.push(adapterResult.phase);
```
