### `src/services/benchmarkDto.ts`（60 行）

|------|------|

| `summarizeBenchmarkRunResult(...)` | 提取 totalCases, passed/failed/skipped/errored counts, supported/unsupported categories |

### `src/app/api/benchmark/report/route.ts`（36 行）

**GET /api/benchmark/report**

- 只读 — 不修改任何状态
- 不需要 API key
- 不调用 LLM
- 运行所有 firstReplayBenchmarkFixtures
- 返回结构化 benchmark report

### `tests/app/benchmarkReportRoute.test.ts`（8 tests）

| 测试 | 验证 |
|------|------|
| GET returns 200 | HTTP status OK |
| returns valid report with summary | generatedAt, summary, results |
| correct pass/fail/skip counts | passedCount ≥ 3, skippedCount ≥ 1 |
| results include supported categories | memory_decay, homeostasis_recovery, belief_evolution |
| passed results have assertion counts | passedAssertions > 0, durationMs > 0 |
| supported/unsupported categories | supported list correct, unsupported non-empty |
| reasons and warnings present | reasons > 0, warnings > 0 |
| idempotent — no mutation | two calls produce same structure |
