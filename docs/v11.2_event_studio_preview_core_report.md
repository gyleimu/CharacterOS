## 2. Three Preview Modes

| Mode | What It Does | Mutates State | requiresConfirmation |
|------|-------------|--------------|---------------------|
| **parse_only** | Parse event text вҶ’ category, emotion, intensity | вқҢ No | false |
| **impact_preview** | Parse + estimate impact on all channels | вқҢ No | false |
| **full_preview** | Clone state, simulate event, run Reality Audit, compute deltas | вқҢ No (clone only) | **true** |

## 6. State Immutability

Every test that runs `buildEventStudioPreview` verifies:
- Serialize baseline state before preview
- Serialize baseline state after preview
- Assert identical

The clone is created via `serialize вҶ’ deserialize` and never touches the original state object.
