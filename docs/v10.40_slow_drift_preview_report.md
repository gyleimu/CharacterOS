## 2. Animation Design

### rAF Loop
```
requestAnimationFrame(animLoop)
  → if (!paused) render(now)
  → requestAnimationFrame(animLoop)
```
Single loop. No timers. No setInterval. Page not visible → browser throttles naturally.

### Drift Math
```
period = 50 + hash(nodeId + "per") * 50     // 50-100 seconds per cycle
phase  = hash(nodeId + "phase") * 2PI       // per-node phase offset

offsetX = cos(omega * t + phase) * driftMag * cos(driftAngle)
offsetY = sin(omega * t + phase) * driftMag * sin(driftAngle)

screenOffset = offset * driftVisibility * scale
```

- **Period**: 50–100 seconds → extremely slow
- **Amplitude**: 0.15–0.55 world units → <0.5px at L0, ~2–4px at L4
- **Deterministic**: `hash(nodeId)` for angle, period, phase. No `Math.random()`

### Drift Visibility Ramp
```
driftVisibility = clamp((scale - 5.0) / (5.6 - 5.0), 0, 1)
```
- scale < 5.0 (L3-): visibility = 0 → no drift
- scale 5.0–5.6 (L4 transition): visibility ramps 0→1
- scale > 5.6 (L4+): visibility = 1 → full drift

### Label Stability
Labels follow node drift at **50% offset** to reduce visual jitter while maintaining association with their node.

## 3. Controls

| Button | Action |
|--------|--------|
| **Pause** | Toggle animation loop. Freezes drift at current position. Text changes to "Play" when paused |
| **Motion** | Toggle Reduced Motion. When active: driftVis=0, all nodes static, drift vectors hidden. Text changes to "Motion:off" |
| **Reset** | Jump to L0 (scale=0.5), re-center viewport |
| **L0-L4** | Jump to band center scale, re-center |

## 5. Visual Feel

- **L0-L2**: Completely static. No visible drift
- **L3**: Still static (driftVis = 0)
- **L4 transition (scale 5.0–5.6)**: Nodes begin to shift very slightly. Barely perceptible
- **L4 full (scale 5.6–8.0)**: Nodes show slow elliptical oscillation. Period 50–100 seconds. Amplitude 2–4 screen pixels. Drift vectors point in the direction of movement
- **Reduced Motion**: All drift hidden. Clean static view at all zoom levels

## 9. Known Limitations

- Drift vectors are computed once per frame from current time — no smoothing between frames at very low frame rates
- No touch support (mobile)
- No pinch-to-zoom
- Labels at 50% drift offset may still show slight jitter at extreme zoom (scale > 7)
- Page visibility API not used — browser throttles rAF naturally when tab is hidden
- No frame rate independent drift speed normalization — drift speed is wall-clock based and independent of frame rate
