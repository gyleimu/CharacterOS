# Character Physics Notebook

这个笔记本收集的是那些很难被简单 prompt 规则解释清楚的问题。

它们不只是心理学问题，而是 Character Physics 的模拟问题：

一个人为什么会漂移成现在这个样子？

同样的事件，为什么会在不同人身上生成完全不同的性格后果？

## 星云模型的底层结构

CharacterOS 的 Personality Galaxy 不应该只从 Personality Core 开始。人格核心不是最底层，它建立在更深的生命驱动力和承压结构之上。

更完整的底层链路应该是：

```text
Biological Nature（生物本质）
        ↓
Psychological Boundary（心理边界）
        ↓
Personality Core（人格核心）
        ↓
Belief System（信念）
        ↓
Need Deficiency（缺失）
        ↓
Desire（欲望）
        ↓
Behavior（行为）
        ↓
Experience（经历）
        ↓
Memory Galaxy（记忆星系）
        ↓
Personality Drift（人格漂移）
```

这条链路不是线性因果的唯一方向，而是 Character Physics 的主循环。人物不是先有一个静态人格，然后被事件改变；人物是一个生命系统：生物本质提供底层驱动力，心理边界决定承压容量，人格核心决定倾向，信念解释世界，缺失制造张力，欲望推动行为，行为带来经历，经历沉积为记忆星系，记忆星系再长期牵引人格漂移。

### Biological Nature（生物本质）

生物本质位于人格之下，是更稳定、更难改变的底层常数。它不是“性格好坏”，而是生命系统的基础驱动力。

当前可建模维度包括：

```text
survival              生存本能
selfPreservation      自我保护
selfInterest          自利倾向
reproduction          繁衍倾向
attachment            依恋需求
belonging             归属需求
statusSeeking         地位竞争
control               控制需求
curiosity             好奇心
imagination           幻想能力
painAvoidance         避痛倾向
rewardSeeking         奖赏追求
```

它们解释的是：

```text
为什么人物会被某类刺激更强烈地牵动。
为什么同样事件对不同人压力不同。
为什么有些欲望和恐惧很难被理性说服。
```

### Psychological Boundary（心理边界）

心理边界决定人物能承受多少压力、创伤和长期影响。人格核心决定方向，心理边界决定容量。

边界不是硬壳，而是弹性结构：

```text
capacity      容量
resilience    韧性
integrity     完整度
stressLoad    当前压力负载
cracks        裂纹
recoveryRate  恢复速度
phase         stable / strained / overflow
```

当压力没有超过边界时，系统维持稳定；当压力接近边界时，人物进入 strained；当压力超过边界时，发生 Boundary Overflow。Overflow 不必然意味着崩溃，它可能通向崩溃、冷漠、防御、偏执、成长、幽默化、野心化，或者某种新的稳定状态。

### Personality Core（人格核心）

人格核心是人物在高维人格空间中的位置和速度：

```text
position
velocity
learningRate
momentumAlpha
```

人格核心不是静态 Profile。它会被 Memory Galaxy 的长期合力牵引，但必须缓慢漂移。单个事件不应该瞬间改写人格。

### Belief / Need / Desire / Behavior

信念不是标签，而是记忆和解释方式沉淀出的世界模型。

缺失不是欲望本身，而是内部张力。

欲望是缺失为了降低张力而产生的方向。

行为不是剧情选择，而是当前人格、信念、缺失、欲望和边界压力共同作用后的输出。

### Experience / Memory Galaxy / Personality Drift

行为带来经历，经历不会直接改人格，而是形成 ImpactParticle 和 MemoryNode。

相似记忆形成 ImpactCluster。星团拥有质量、密度、稳定度和引力。长期重复经历形成稳定星团，星团通过慢变量牵引人格核心：

```text
P(t+1) = P(t) + velocity(t+1)
velocity(t+1) = alpha * velocity(t) + clusterForce * learningRate
```

因此，CharacterOS 的目标不是让 AI 编故事，而是模拟一个生命系统如何在本能、边界、人格、信念、行为和记忆星系中持续演化。

## 核心问题

为什么经历相似事件的人，最后会变成完全不同的人？
每个人的初始性格特征是不同的

同样的失去，可能让一个人变成熟，也可能让一个人变冷漠、偏执、幽默、残忍、沉默、野心勃勃，或者彻底崩塌。
这边可以加一个除了性格特征之外的，可以设定成抗压能力，在物理模型里面，每个人的星云图的大小，边界是不同的，超过了边界，就可能会情绪崩溃
关于人格核心与心理边界

在 Character Physics 中，人格核心（Personality Core）和心理边界（Psychological Boundary）应该被视为两个相互独立但共同作用的系统。人格核心决定一个人是谁，例如开放性、外向性、宜人性、神经质、野心、控制欲等特征；而心理边界则决定一个人能够承受多少压力、创伤和长期影响。人格决定方向，边界决定容量，两者共同决定经历最终会把一个人变成什么样。

因此，同样的失去、失败或背叛，并不会让所有人产生相同的结果。失恋可能让一个人变得成熟，也可能让另一个人变得冷漠、偏执、沉默、野心勃勃、幽默化，甚至彻底崩溃。事件本身并不直接决定结果，真正决定结果的是事件、人格、信念、关系以及心理边界共同作用后的产物。

在心理宇宙模型中，每个人都拥有不同大小的心理星云。除了人格核心的位置不同之外，星云本身也具有不同的边界半径。高韧性的人拥有更大的心理边界，能够吸收更多冲击；而脆弱的人则拥有较小的边界，更容易在长期压力积累后进入不稳定状态。当长期压力和影响因子的总和没有超过边界时，系统能够维持稳定；而一旦超过边界，就会发生 Boundary Overflow，使整个系统进入新的阶段。

心理边界并不是刚性的外壳，而更像是一种具有弹性和完整度的结构。重大创伤会在边界上留下裂纹，使完整度下降，未来原本微小的事件也可能引发巨大的情绪反应，这与现实中的创伤后应激障碍（PTSD）具有一定相似性。同时，边界也具有恢复能力，时间、安全感、友谊、爱情、成功以及治疗都能够缓慢修复边界，提高系统的稳定性。

值得注意的是，边界被突破并不一定意味着崩溃。长期压力积累后，系统可能发生一种类似于相变（Phase Transition）的过程。在连续的痛苦、压抑或失败之后，人可能会在某一天突然“想通”，进入一种新的稳定状态；也可能彻底崩塌，或者朝着成长、冷漠、防御、偏执、野心、幽默化等不同方向演化。因此，创伤本身不会定义一个人，真正定义一个人的，是人格核心、心理边界、信念结构以及恢复能力之间复杂而动态的相互作用。

或许，对于 Character Physics 来说，一个人物不应该被视为一个静止的点，也不应该被视为几项固定属性，而应该被视为一个拥有核心、星团、边界、裂纹、恢复能力和世界线，并随着时间不断演化的心理宇宙。

Character Physics 不应该用一条直接规则回答这个问题。

错误方式：

```text
if 失恋:
    信任 -= 0.2
```

更合理的路径应该是：

```text
事件
↓
解释方式
↓
情绪
↓
记忆节点
↓
记忆簇
↓
信念
↓
需求 / 恐惧
↓
行为模式
↓
人格漂移
```

- 对自己有利，人本是自私的。  关于人格之下的生物本质

随着 Character Physics 理论逐渐完善，我开始意识到，人格特征、信念系统、需求系统和欲望系统可能都不是最底层的结构。在这些结构之下，还存在一层更加稳定、更难改变的底层驱动力，它们来源于生物演化本身，可以被视为角色的“生物本质（Biological Nature）”。

人格会变化，信念会改变，欲望会转移，但生物本能通常具有极高的稳定性。它们并不是某种性格特征，而更像是一组深层常数，持续影响着角色的一切行为和心理活动。

这些底层驱动力可能包括：

生存本能（Survival）
自我保护（Self-preservation）
自私倾向（Self-interest）
繁衍倾向（Reproduction）
亲密关系需求（Attachment）
社会归属需求（Belonging）
地位竞争（Status Seeking）
控制欲（Control）
好奇心（Curiosity）
幻想能力（Imagination）
逃避痛苦与追求快乐（Pain Avoidance / Reward Seeking）

其中，幻想能力是一个非常特殊的部分。人类并不仅仅生活在现实世界中，还生活在自己的心理世界之中。人们会回忆过去、幻想未来、进行自我对话、构建理想人格、想象未发生的事件，甚至通过幻想缓冲现实带来的痛苦。因此，幻想并不是错误，而是一种重要的心理机制，它能够带来希望，也可能导致逃避现实。

在 Character Physics 中，这些生物本质不应被视为人格的一部分，而应位于人格之下。它们更像是整个心理宇宙的底层物理常数。人格核心决定“他是谁”，心理边界决定“他能承受多少”，而生物本质则决定“他为什么会这样存在”。

因此，一个角色的形成过程或许应该重新描述为：

Biological Nature
→ Personality Core
→ Belief System
→ Need Deficiency
→ Desire System
→ Behavior
→ Repeated Experience
→ Personality Drift

也就是说，人物的行为并不完全来源于人格，而人格本身也建立在更深层的生物本能之上。Character Physics 最终模拟的，或许并不是一个“拥有性格的人”，而是一个在生物本能、经历、信念和时间共同作用下不断演化的生命系统。
- 为什么有些人一旦“对过一次”，就再也无法怀疑自己？
- 为什么有些人宁愿选择痛苦的确定性，也不愿接受不确定的希望？
- 为什么恐惧有时候会伪装成聪明？
- 为什么有些人会把脆弱理解成陷阱？
- 为什么有些人会忠诚于自己的伤口？
- 为什么控制感对某些人来说等于安全感？
- 为什么有些人在被反驳时反而会变得更极端？
- 为什么孤独会让一些人更清醒，却让另一些人更扭曲？
- 为什么人的内心有时会通过缩小世界来保护自己？
- 为什么有些人选择怀疑，是因为信任会让他们感觉自己很弱？

## 建模提示

这些问题说明，Character Physics 不能只依赖简单的人格数值变化。

真正重要的是隐藏变量：

- 解释风格
- 依恋策略
- 羞耻敏感度
- 控制需求
- 信任基线
- 威胁识别偏差
- 自我故事僵化程度
- 记忆聚类倾向
- 修复能力
- 社会归属需求
- 主体性感受
- 情绪消化速度
- 身份弹性
- 道德抑制力
- 习得性无助
- 意义生成能力

这些问题也可以变成测试用例：

```text
给定相似的 EventImpactVector，
当角色拥有不同的记忆簇、信念先验和需求缺口，
那么长期人格漂移应该产生不同方向。
```


角色不仅有 Psychological Boundary，也应该有 Behavioral Boundary。

Psychological Boundary 决定角色能承受多少心理压力；Behavioral Boundary 决定角色在正常状态下行为的极限范围。一个角色即使愤怒，也不一定会做出极端行为，因为他的人格、道德感、控制力、恐惧、社会规则感都会限制他的行为。

但当事件刺激超过角色的承受范围，或者情绪强度超过行为边界时，角色可能发生 Behavioral Boundary Break。此时角色会做出平时不会做、甚至自己事后也难以解释的行为。这种行为不是完全随机，而是由当前情绪、长期压抑、核心信念、心理裂纹和生物本能共同推出来的失控结果。

因此，行为系统应该区分两种状态：

Normal Behavior Range：正常行为范围。
Boundary Break Behavior：突破边界后的异常行为。

公式上可以先简化为：

emotionPressure = emotionIntensity × eventImpact × accumulatedStress

如果：

emotionPressure < behaviorBoundary

则角色只能在正常行为范围内选择行为。

如果：

emotionPressure > behaviorBoundary

则触发 Behavior Boundary Break，允许出现超出平时设定的行为。

这可以解释为什么同一个人平时克制、理性、温和，但在长期刺激、重大打击或无法忍受的情境下，会突然做出完全超出原本人设的事情。
