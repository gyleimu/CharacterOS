## 1. 新增文件

### `src/core/graph/mindGraphTypes.ts`（250 行）

**最小类型系统 + pure helper functions。不构建 graph，不读 state。**

#### 常量

| 常量 | 数量 | 说明 |
|------|------|------|
| `MIND_GRAPH_NODE_TYPES` | 11 | personality_core … benchmark_signal |
| `MIND_GRAPH_EDGE_TYPES` | 14 | belongs_to_cluster … derived_from |
| `MIND_GRAPH_EVIDENCE_SOURCES` | 5 | state, trace, subprocess_trace, benchmark_result, derived |

#### 接口

| 接口 | 字段 |
|------|------|
| `MindGraphNode` | id, type, label, source, stableId, weight?, mass?, risk?, confidence?, metadata |
| `MindGraphEdge` | id, type, sourceNodeId, targetNodeId, directed, weight?, confidence?, evidence, metadata |
| `MindGraphSummary` | nodeCount, nodeCountsByType, edgeCount, edgeCountsByType, averageEdgeWeight, strongEdgeCount, persistedNodeCount, derivedNodeCount |
| `MindGraphSnapshot` | version, characterId, generatedAt, nodes[], edges[], summary, warnings[], reasons[] |

#### Helper functions

| 函数 | 用途 |
|------|------|
| `isMindGraphNodeType(v)` | Type guard for 11 node types |
| `isMindGraphEdgeType(v)` | Type guard for 14 edge types |
| `createMindGraphNodeId(type, stableId)` | Deterministic: `"memory:mem_001"` |
| `createMindGraphEdgeId(type, source, target)` | Deterministic: `"activates_belief:n1→n2"` |
| `summarizeMindGraph({ nodes, edges })` | Computes MindGraphSummary without mutation |
