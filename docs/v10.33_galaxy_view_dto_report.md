## 5. Drift Vector Design

- **Deterministic**: `deterministicAngle(seed)` based on `driftSeedPrefix:nodeId:drift`
- **No Math.random()**: All values derived from deterministic hash
- **No Date.now()**: Drift is a pure function of node identity + graph structure
- **Low magnitude**: 0.05–2.0 logical pixels per frame. Most nodes < 1.0
- **Classified direction**: `strengthening | weakening | approaching_core | drifting_outward | stable`
- **Human-readable reason**: e.g., "memory shows slow inward drift toward personality core"
- **Configurable**: `computeDrift: false` disables all drift; `driftSeedPrefix` changes seed

---

## 6. Hover Summary Design

Every view node carries a `hover` field with:

```typescript
{
  nodeId,           // from graph
  label,            // truncated to 80 chars
  nodeType,         // from graph
  risk,             // high | medium | low | undefined
  weight,           // [0, 1], rounded to 4 decimals
  primaryReason,    // one-sentence derived from connected edges
  edgeCount,        // number of connected edges
  evidenceSourceCount, // distinct evidence sources
  driftSummary      // present only when drift exists
}
```

**No raw state exposure**: Hover summaries contain no coordinate values, no cluster internals, no serialized state, no CharacterPhysicsState references.
