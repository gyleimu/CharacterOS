## 1. Executive Summary

V10.37 is the integration point for the Mind Galaxy Viewer data pipeline. It composes all four V10.33–V10.36 modules into a single `GalaxyFrame` at any point in time. One function call produces everything a future Canvas 2D renderer needs to draw one frame.

**Pipeline:**
```
MindGalaxyViewSnapshot
  → buildGalaxyZoomState(scale)           [V10.35]
  → buildGalaxyDriftAnimationSnapshot()   [V10.36]
  → sampleGalaxyDriftSnapshot(timeMs)     [V10.36]
  → buildGalaxyCanvasRenderCommands()     [V10.34]
  → applyDriftFrameToCommands()           [V10.36]
  → GalaxyFrame
```

**New files:**
- `src/core/graph/mindGalaxyFrameComposer.ts` — 250 lines
- `tests/core/graph/mindGalaxyFrameComposer.test.ts` — 30 tests

## 4. Composition Pipeline

```
composeGalaxyFrame(snapshot, { scale: 4.0, timeMs: 5000, ... })

Step 1: resolveOptions()     → ResolvedGalaxyFrameOptions
Step 2: buildGalaxyZoomState(scale)  → GalaxyZoomState (L3, bandProgress 0.5)
Step 3: buildGalaxyDriftAnimationSnapshot(snapshot, zoomState, opts)
                                → GalaxyDriftAnimationSnapshot
Step 4: buildGalaxyCanvasRenderCommands(snapshot, renderOpts)
                                → GalaxyCanvasRenderCommand[] (static)
Step 5: sampleGalaxyDriftSnapshot(animation, timeMs)
                                → Map<nodeId, GalaxyDriftFrame>
Step 6: applyDriftFrameToCommands(staticCommands, driftFrame)
                                → GalaxyCanvasRenderCommand[] (with offsets)
Step 7: build summaries          → GalaxyFrameSummary + GalaxyFrameLayerSummary[]
       → GalaxyFrame
```
