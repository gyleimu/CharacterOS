# CharacterOS V3.1 Reward Dopamine Foundation Report

## Version

```text
CharacterOS V3.1 Reward Dopamine Foundation
```

## Goal

V3.1 introduces the first Reward / Dopamine system.

The goal is to move CharacterOS away from goal-machine behavior and toward reward-driven human behavior.

```text

-> Desire
-> Action
-> Reward
-> Adaptation
-> New Desire
```

## Event Integration

`CharacterPhysicsEngine.processEvent()` now creates a reward input from event category:

```text
support      -> attachment reward
success      -> achievement reward
abandonment  -> relief / harmful habit reward
betrayal     -> relief / harmful habit reward
general      -> habit reward
```

The result is stored as:

```text
rewardResult
```

and `state.rewardState` is updated.

## Continuous Tick Integration

`runContinuousTick()` now recovers reward state toward baseline:

```text
dopamineLevel
dopamineThreshold
rewardSensitivity
noveltyNeed
craving
```

The trace now includes:

```text

rewardAfter
```
