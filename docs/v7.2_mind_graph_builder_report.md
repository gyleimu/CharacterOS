## 1. 新增文件

### `src/core/graph/mindGraphBuilder.ts`（300 行）

**从 CharacterPhysicsState 构建 MindGraphSnapshot。Pure builder。**

| 函数 | 用途 |
|------|------|
| `buildMindGraphSnapshot(state, options?)` | 从 state 构建完整 graph snapshot |

#### 第一版覆盖的节点类型（5/11）

| Node type | Source | Count |
|-----------|--------|-------|
| `personality_core` | state.coordinate + state.identity | 1 |
| `memory` | state.memories | state.memories.length |
| `impact_cluster` | state.clusters | state.clusters.size |
| `belief` | state.beliefStates (filtered >= 0.02 strength) | variable |
| `need` | deriveNeedDeficiencies (filtered >= 0.15 intensity) | variable |

#### 第一版覆盖的边类型（5/14）

| Edge type | From | To |
|-----------|------|-----|
| `belongs_to_cluster` | memory | impact_cluster (by memory.clusterId) |
| `pulls_personality` | impact_cluster | personality_core |
| `activates_belief` | memory | belief (by belief.sourceMemoryIds) |
| `creates_need` | belief | need |
| `derived_from` | need | personality_core |

#### 暂不覆盖（V7.3+）

| Type | 原因 |
|------|------|
| impact_particle | 需要 trace-based construction |
| desire | 需要 deriveDesires |
| behavior_bias | 需要 deriveBehaviorBiases + deriveCharacterState |
| temporal_process | 需要 ContinuousTickTrace |
| internal_state_variable | 需要 InternalStateFieldSnapshot |
| benchmark_signal | 需要 BenchmarkResult |
