## Dashboard Behavior

The Continuous Tick panel now displays Homeostasis hints after Homeostasis reasons.

These hints help human inspection answer:

```text
is pressure too high?
is change resistance too strong?
is scar retention shaping recovery?
is boundary still overflowing after regulation?
is there a recovery window?
```
