## Architecture Meaning

CharacterOS now treats manual calibration as a governed process.

The system can say:

```text
You may continue.
Slow down and observe.
Stop changing this character for now.
```

This protects the deeper principle:

Characters should evolve, but they should not be over-edited.

Manual control exists, but it now has restraint.
