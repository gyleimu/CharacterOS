## 2. How Timeline Differs from LLM Explanation

| LLM Explanation | V11.5 Timeline |
|----------------|-----------------|
| Generates natural language prose | Produces structured causal steps |
| May invent plausible-sounding causes | Only uses evidence from audit/event history |
| Cannot trace to source | Every step has `sourceRef` to audit entry |
| Ungrounded claims possible | `grounded: false` when no direct evidence |
| Confidence is opaque | Explicit confidence per step + overall |

## 3. Causal Chain Structure

```
Event (audit entry)
  → Memory (memory formation)
    → Belief (belief adjustment)
      → Personality (coordinate delta)
        → Boundary (stress state)
          → Current Surface (emotional/behavioral state)
```

Each step is a `CausalStep` with:
- `stepId` — unique identifier
- `type` — event/memory/belief/need/personality/boundary/decision
- `sourceRef` — links to audit entry or "current_surface"
- `direction` — increased/decreased/stabilized/triggered
- `magnitudeBand` — high/moderate/low
- `confidence` — 0–1 per step
- `grounded` — true if sourced from audit, false if from surface

## 4. When Evidence Is Missing

| Scenario | Behavior |
|----------|----------|
| No audit history, no recent events | `confidence: "low"`, `groundingStatus: "ungrounded"`, warnings |
| Some history but gaps | `confidence: "moderate"`, `groundingStatus: "partially_grounded"` |
| Rich history, all steps grounded | `confidence: "high"`, `groundingStatus: "grounded"` |
| Steps from "current_surface" | `grounded: false` (descriptive, not causal) |

## 5. Focus Filtering

| Focus | Filters To |
|-------|-----------|
| `emotion` | event + memory + boundary |
| `belief` | event + belief |
| `need` | event + need |
| `desire` | event + desire |
| `personality` | event + personality + belief |
| `decision` | event + decision |
| `stress` | event + boundary |

## 7. Current Limitations

- Needs/Demands not yet traced through DerivedState → `CausalStep`
- Decision-to-action chain not traced (no behavior decision history)
- Time range comparison only structural — no statistical trend
- "current_surface" steps are descriptive, not causal (marked `grounded: false`)
