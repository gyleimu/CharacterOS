## 2. Data Export Script

### Usage

```bash
npx tsx scripts/export-mind-galaxy-preview-data.ts
```

### Pipeline

```
createLinFanBlueprint() → CharacterPhysicsState (seeded)
  → buildMindGraphSnapshot(state) → MindGraphSnapshot
  → buildGraphLayoutSnapshot(graph) → GraphLayoutSnapshot
  → buildMindGalaxyViewSnapshot(graph, layout) → MindGalaxyViewSnapshot
  → preview-compatible JSON
```

### Output Format

```json
{
  "version": "10.42.0",
  "generatedAt": "2026-06-28T...",
  "characterId": "lin-fan",
  "centerX": 500, "centerY": 500,
  "totalNodes": 40, "totalEdges": 48,
  "nodes": [
    { "id": "personality_core:...", "type": "personality_core",
      "label": "Personality Core", "x": 500, "y": 500,
      "r": 28, "fill": "#FFD700", "stroke": "#B8860B",
      "w": 1.0, "z": 100, "drift": { "a": 4.2, "m": 0.35 } }
  ],
  "edges": [
    { "id": "belongs_to_cluster:...", "t": "belongs_to_cluster",
      "s": "memory:...", "d": "impact_cluster:...",
      "w": 0.8, "style": { "stroke": "#4169E1", "sw": 1.5, "op": 0.6 },
      "dir": 0 }
  ]
}
```

### What it does NOT contain

- No `CharacterPhysicsState` — only graph-projected data
- No raw `clusters` Map
- No `proceduralRoutines` dump
- No `galaxyState`
- No `finalState`, `serializedState`
- No full memory content — only labels (truncated to 40 chars)

## 3. Preview Integration

### Data Sources

| Source | Nodes | Edges | How |
|--------|-------|-------|-----|
| **Fixture** (default) | 13 | 10 | Hand-crafted small galaxy — always works offline |
| **Real Data** | 40 | 48 | Generated from CharacterOS blueprint — click "Real" button |

### How to Switch

1. Open `previews/mind-galaxy-preview.html`
2. The HUD shows `fixture` initially
3. Click the **Real** button to switch to real data
4. If real data hasn't been loaded (e.g., file:// blocks fetch), a message appears in the browser console
5. Click **Fixture** to switch back

### file:// Limitation

When opened via `file://`, browsers may block `fetch()` to local JSON files due to CORS/same-origin policies. In this case:

- **Option A**: Open via a local static server:
  ```bash
  npx serve previews/
  # or
  cd previews && python -m http.server 8080
  ```
  Then open `http://localhost:8080/mind-galaxy-preview.html`

- **Option B**: Use the built-in fixture data (always works, no server needed)
