## 1. Executive Summary

V10.39 upgrades the static preview to an interactive zoomable view. Mouse wheel zooms in/out centered on cursor. Drag pans the viewport. Zoom levels transition smoothly with opacity fades — no hard cuts. The V10.35 zoom band math is inlined for continuous LOD control.

## 3. Interaction Model

```
Wheel up      → zoom in  (scale × 1.12, centered on cursor)
Wheel down    → zoom out (scale × 0.89, centered on cursor)
Left drag     → pan (world moves under viewport)
L0-L4 buttons → jump to band center
Reset button  → jump to L0 (scale 0.5)
```

---

## 4. LOD Fade Behavior

Using the V10.35 zoom bands and 20% transition zones:

```
Scale 0.25–0.35 (L0 fade):  core fades from 0→1
Scale 0.75–0.90 (L1 fade):  clusters + temporal_process fade in
Scale 1.5–1.80  (L2 fade):  memory/belief/need/desire fade in
Scale 3.0–3.40  (L3 fade):  behavior_bias + labels appear, edge labels become visible
Scale 5.0–5.60  (L4 fade):  drift vectors fade in
```

Opacity is continuous — scrolling through a band smoothly reveals the new layer's elements.

---

## 5. Known Limitations

- **No animation loop**: drift vectors are sampled once per render, not continuously animated. Refresh to see new positions
- **No touch support**: desktop mouse only
- **No pinch zoom**: single-finger pan, wheel zoom only
- **Center-zoom approximation**: cursor-centered zoom uses `worldUnderCursor → scale → recompute pan`, which is mathematically correct for the first-order approximation. For extreme zoom-in near edges, slight cursor drift may occur
- **No inertia**: pan stops immediately on mouse release — intentional simplicity
- **No mobile**: not touch-optimized
