## Manual Review Checklist

Before handing off, run through this checklist:

- [ ] Open `index.html` in a desktop browser
- [ ] Click through all 6 tabs in order
- [ ] Test Review Mode toggle
- [ ] Test scenario filter buttons (All / 关系确认 / 机会 / 纠偏 / 控制)
- [ ] Expand a scenario action with "展开完整行动"
- [ ] Open Mind Galaxy tab (confirm lazy load)
- [ ] Click "全屏打开" for galaxy
- [ ] Test keyboard navigation (ArrowLeft/Right, Home, End)
- [ ] Verify URL hash changes on tab switch
- [ ] Print preview (Ctrl+P) — confirm readable
- [ ] Read README.md — confirm all links/paths work

## Known Limitations

| Limitation | Notes |
|---|---|
| Galaxy must be pre-generated | `export-mind-galaxy-static-artifact.ts` must run first |
| Single character only | 林凡 is the only demo character |
| Static data | Same seed always produces same output |
| No persistence | Refresh resets Review Mode and filters (tabs survive via hash) |
| Desktop browser only | Not optimized for mobile viewports |
| English/CJK mix | Some labels are English (emotion names, strategy IDs), most content is Chinese |
