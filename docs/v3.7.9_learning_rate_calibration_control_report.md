# CharacterOS V3.7.9 Learning Rate Calibration Control Report

## Purpose

Expose the first actual physics parameter for human-in-the-loop replay tuning.

Previous manual controls adjusted calibration standards.

This pass adds:

```text
learningRate
```

to replay calibration.

## What It Controls

`learningRate` controls how strongly replay cluster force affects personality velocity during calibration runs.

It does not directly mutate the live dashboard character state.

It only affects:

```text
fixed replay simulation
calibration results
Dashboard calibration panel
```

## API

`GET /api/trace/replay/calibration` now supports:

```text
?learningRate=0.06
```

Safe range:

```text
0.001 ~ 0.2
```

## Dashboard

The Replay 校准 panel now includes:

```text
学习率
```

Changing it and clicking:

```text
重新校准
```

reruns calibration with the selected learning rate.
