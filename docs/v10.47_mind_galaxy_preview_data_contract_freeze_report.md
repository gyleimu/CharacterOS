## 4. Validation Rules (37 checks)

### Root object
- Must be non-null object
- 12 required top-level keys
- 9 forbidden raw-state keys (finalState, CharacterPhysicsState, serializedState, coordinate, proceduralRoutines, etc.)

### Scalars
- version: non-empty string
- generatedAt: non-empty string
- characterId: non-empty string
- centerX/Y: finite numbers
- totalNodes/Edges: non-negative integers
- driftNodeCount: non-negative integer

### Nodes (per node)
- Required keys: id, type, label, x, y, r, fill, stroke, w, z
- id: non-empty, unique
- type: non-empty string
- x, y, r: finite, r >= 0
- w: [0, 1] (warning if out of range)
- z: integer
- drift.a: finite; drift.m: >= 0
- No forbidden keys in node (serializedState, rawCluster, coordinate)

### Edges (per edge)
- Required keys: id, t, s, d, w, style, dir
- id: non-empty, unique
- s, d: exist in nodes (warning only — allows forward-compat)
- dir: 0 or 1
- style.stroke: string; style.sw: >= 0; style.op: [0, 1] (warning)

### Counts
- totalNodes === nodes.length
- totalEdges === edges.length
- driftNodeCount ≈ actual drift count (warning if mismatch)
