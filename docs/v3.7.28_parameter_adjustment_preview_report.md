# CharacterOS V3.7.28 Parameter Adjustment Preview Report

This update adds a Parameter Adjustment Preview layer.

The system can now show what a manual adjustment draft would do, while still refusing to mutate the character automatically.

## Preview Output

Each preview item contains:

```text
targetPath
currentValue
draftSuggestedValue
previewValue
cappedDelta
risk
status
reasons
```

## Safety Rule

Preview values are capped by a small maximum delta.

Default:

```text
maxDelta = 0.025
```

This means even a strong draft cannot become a large instantaneous change.

## Design Constraint

Adjustment Preview is still read-only.

It does not apply patches.
It does not persist changes.
It does not rewrite personality.

## Architecture Meaning

This is the next step toward operator-assisted calibration.

The workflow is now:

```text
observe
accumulate
draft
preview
human review
```

CharacterOS can make possible changes visible without letting the system destabilize itself.
