| **Core Reality Gate** | `runCoreRealityRegressionGate()` | Reality audit, long-term accumulation, event type coverage |

### 2.3 Gate Rules

| Level | Condition |
|-------|-----------|
| **FAIL** | Benchmark pass rate < 80%, OR benchmark failures > 1, OR reality gate FAIL, OR any unified failure |
| **WARN** | Benchmark pass rate within 3% of threshold, OR reality gate WARN, OR documented warnings exist |
| **PASS** | All benchmarks pass, reality gate has ≤ allowed warnings, no failures |

## 5. Usage

```bash
# Full unified quality gate
npx tsx scripts/run-unified-quality-gate.ts

# Skip benchmark for faster iterations
npx tsx scripts/run-unified-quality-gate.ts --skip-benchmark

# Run just reality gate
npx tsx scripts/run-core-reality-gate.ts

# Run all targeted tests
npx vitest run tests/core/audit/unifiedQualityGate.test.ts \
  tests/core/audit/coreRealityRegressionGate.test.ts \
  tests/core/audit/realityAudit.test.ts \
  tests/core/audit/eventTypeCoverageAudit.test.ts \
  tests/core/audit/impactCalibrationAudit.test.ts
```

Exit code: 0 for PASS/WARN, 1 for FAIL (CI-compatible).
